This is the first and static version of this site. It is designed partly upon Bootstrap 4, and on it's jQuery module. The navigation panel is coded in JS. Other modules used are Font Awesome and Line Awesome, as well as gfx resources from Unsplash and Pexels.

The page is planned to be reproduced at least twice, one built upon a PHP version, or another back-end solution, and another into a WordPress theme.

The login solutions will obviously not work on this version, as is not connected to any type of database or server side module. Page has only been tested on Chrome, Edge, and IE11.

The idea behind the navigation pane is a flexible unit which can easly be adjusted to one's needs, and to ultimately become hooked up to some sort of editor on the back-end side.

Even though Bootstrap 4 is used for styling, the javascript module is not dependent on it.

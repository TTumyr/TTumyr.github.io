This is the first and static version of this site. It is based the Bootstrap 4 framework, and partly on it's jQuery module. The left navigation pane is coded in JS. Other modules used in the creation are Font Awesome and Line Awesome, as well as gfx resources from Unsplash and Pexels.

This site is planned in at least two more versions, one built upon a PHP version and another one integrated into a WordPress theme. It is likely that other versions will be tested as well, like NodeJS.

The login solutions will obviously not work as this version is not connected to any type of database, or server side module.

The idea behind the navigation pane is a flexible unit which can easly be adjusted to one's needs, and to ultimately become hooked up to some sort of editor on the server side.

Even though Bootstrap 4 is used on the styling, the .js module is in itself not dependent on it.

You can download the source code here.

At the moment this is a private site still under development, and if you are reading this you are likely connected privately to the project. I would love to hear about your experiences with the functionality of the site, and how to improve it.
